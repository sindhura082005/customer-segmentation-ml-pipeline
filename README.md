### Customer Segmentation & Analytics Platform
An end-to-end customer segmentation project built using Python, Machine Learning, and Streamlit. This application allows users to upload customer data and automatically segment customers into meaningful groups using a pre-trained K-Means clustering model, complete with visualuggular Analytics and PCA-based visualizations.

This project demonstrates the complete data analytics lifecycle: data preprocessing, feature engineering, model training, and deployment as an interactive web app.
